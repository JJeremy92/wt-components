# Upgrade guide

Care has been taken to ensure the upgrade process from wtg-material-ui and SUPPLY 0.5 to SUPPLY 1.0 is as easy as possible. However, not all properties or components have matching support in SUPPLY 1.0. This page contains a detailed list of breaking changes and the steps required to upgrade your application to SUPPLY 1.0.

<p>
    <strong>⚠️ Note:</strong> This page is incomplete. Please check back later for more information. If you have additional questions, reach out to us via the support channels.
</p>
